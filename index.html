Projetos

Project Shopping Cart

Simulador de um carrinho de compras, onde podemos adicionar o item para o carrinho e vamos acompanhando o valor total. Foram usadas 3 API's para este projeto.

Project Trybewarts

Formulário simulando uma possível entrada fictícia para escola de magia e bruxaria de Hogwarts. Neste projeto foi práticado as funcionalidades de formulários, usando diversos tipos de input's.

Project Casa de Câmbio

Projeto realizado para as realizações de práticas relacionadas a requisições (fetch) para API's, juntamente com a capacidade de usar bibliotecas e bundling as mesmas para a produção.

Project TrybeTunes

Projeto com funcionalidades de requerimento para a API do Itunes com o objetivo de simular uma plataforma de streaming de musica, desde o login ate as musicas favoritas.
